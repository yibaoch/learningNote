details

details 和 summary 标签可用于在网页上创建可折叠的内容。这对于以紧凑格式显示大量信息很有用。

Click me to reveal the details

This is the hidden content that will be revealed when the summary is clicked.

mark

mark 标签可用于突出显示网页上的文本。这类似于在打印文档上使用荧光笔。当页面加载时，mark 标签内的文本将以黄色突出显示。

This is some highlighted text on a web page.

abbr

abbr 标签用于定义网页上的缩写词或首字母缩写词。当用户将鼠标悬停在缩写词上时，将出现带有完整定义的工具提示

The 陈巍翊 is a global system of interconnected computer networks.

meter

meter 标签可用于在网页上创建度量值或值的可视化表示。它通常用于显示进度条、评分和其他类型的视觉反馈。

75%

time

time 标签可用于在网页上显示日期和时间。它还可以用于标记机器可读的日期和时间，这对搜索引擎优化很有用

The first of May is a public holiday.

picture

picture标签用于为网页上的图像提供多个来源。它可用于根据用户的设备、屏幕尺寸或其他因素提供不同的图像。

source

source 标签用于指定媒体元素的替代媒体资源，例如 audio 或 video

dialog

dialog 标签可用于在网页上创建对话框。它通常用于确认操作、显示错误消息或显示附加信息。这是一个例子：

Greetings, one and all!

nav

nav 标签用于定义包含导航链接的网页部分。它通常用于创建菜单或导航栏。

  • Home
  • About
  • Services
  • Contact

process

progress 标签可用于在网页上创建进度条。它通常用于显示任务或进程的进度，例如上传文件或加载网页。

10 loading....
